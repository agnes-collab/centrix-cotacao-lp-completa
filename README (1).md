# Freitas Centrix — Landing Page Completa

Landing page completa para campanha de geração de leads em Comércio Exterior.

## Como usar

Faça o upload do arquivo `lp-completa.html` renomeado para `index.html` no repositório e ative o GitHub Pages em **Settings → Pages → Branch: main → Save**.

## Formulário

Integrado ao **RD Station** via embed com carregamento assíncrono.

## Conteúdo

- Hero com formulário
- Bloco de dor
- Proposta de valor
- Diferenciais
- Centrix
- Depoimentos
- CTA final
